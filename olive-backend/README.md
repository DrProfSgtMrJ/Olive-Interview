## Setup Instructions:
This project uses [poetry](https://python-poetry.org/) for package management. After installing poetry (at least version `1.8.3`):

`poetry install`

## Run Instructions:
Activate the shell:
```
poetry shell
```
Run fastapi:
```
fastapi dev main.py
```
The application (by default) will start on: `http://127.0.0.1:8000`
